# chainsaw

```
A new web scanner.
Web audit tool.
```

### Download
- Download from [release](releases)
- git clone git@github.com:nic329/chainsaw.git

### Usage
- Linux or Mac
```
./chainsaw http://example.com/
./chainsaw -f url.txt
```
- Windows
```
chainsaw.exe http://example.com/
chainsaw.exe -f url.txt
```
